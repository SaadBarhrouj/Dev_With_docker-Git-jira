# Plateforme de développement web avec Docker pour une collaboration efficace

## Présentation

Ce projet ambitionne de créer une plateforme de développement collaborative pour les applications web exploitant PHP/Laravel côté serveur et Angular côté client. Notre objectif est d'éliminer les disparités d'environnement entre les membres de l'équipe, assurant ainsi une intégration fluide et une cohérence de déploiement.

## Défi

Les disparités de configuration entre les machines des développeurs, notamment les variations de systèmes d'exploitation et de versions d'API, peuvent entraîner des incompatibilités et des retards dans le cycle de développement.

## Solution

Pour répondre à ce défi, nous avons opté pour Docker, une technologie de conteneurisation permettant de créer des environnements isolés et reproductibles. Nous avons conçu une image Docker standardisée, regroupant toutes les dépendances nécessaires pour le développement avec PHP/Laravel et Angular. Cette image est partagée via Docker Hub, garantissant ainsi une uniformité de l'environnement de développement.

Le code source du projet est hébergé sur GitHub, facilitant ainsi la collaboration et le versionnement. Pour la gestion des tâches et la planification des sprints, nous utilisons Jira.

## Technologies utilisées

- **Docker** : Pour la création et le partage d'environnements de développement standardisés.
- **GitHub** : Pour le versionnement du code source et la collaboration sur le projet.
- **Jira** : Pour la gestion des tâches, la planification des sprints et le suivi du projet.
